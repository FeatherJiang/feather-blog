# feather-blog

> My personal blog
> 个人博客

## Website 博客网址

[www.jiangfeather.com](http://www.jiangfeather.com/)

## Technology Stack 技术栈

* React.js
* React Router
* Redux
* Axios

## Build Setup

```
# install dependencies
yarn install

# serve with hot reload at localhost:3000
yarn start

# build for production with minification
yarn build
```
